# データ集計ツール

複数のCSVおよびExcelファイルを読み込み、統合・整形して集計結果を出力するPythonスクリプトです。

## 概要

このスクリプトは以下の処理を実行します：

1. **ファイルチェック**：必要なキーワードを含むファイルの存在確認
2. **データ読み込み**：CSV・Excelファイルの自動判別と読み込み
3. **データクリーニング**：列名の正規化（タブ・改行・「実績」の除去）
4. **データマージ**：年月をキーにして複数ファイルを統合
5. **データ整形**：転置・マルチインデックス化・ソート
6. **結果出力**：`summaryData.csv` に保存

---

## 必要なキーワード

以下のキーワードを含むファイルが期待されます：

- ボイラ
- 構外

- 電力
- ガス

ファイルが不足している場合は警告が表示されますが、ユーザー確認後に処理を続行できます。

## 機能詳細

### 1. ファイルチェック（`check_data_files`）

- 指定ディレクトリ内のファイルをスキャン
- 必要なキーワードの有無を確認
- 不足しているキーワードのリストを返す

```python
all_files_exist, missing = check_data_files(directory)
```

### 2. データ読み込み

- **CSV**: UTF-16 → UTF-8-sig の順で試行
- **Excel**: `pd.read_excel()` で自動判別
- 「構外」を含むファイルは先頭1行をスキップ

### 3. データクリーニング

列名から以下を除去：
- タブ (`\t`)
- 改行 (`\n`)
- 「実績」テキスト
- 前後の空白

```python
df.columns = [str(col).replace('実績', '').replace('\t', '').replace('\n', '').strip() for col in df.columns]
```

### 4. データマージ

- `年` と `月` をキーに外部結合（`how='outer'`）
- 型不一致を防ぐため文字列に統一
- 複数ファイルを段階的に結合

### 5. データ整形

- `年` `月` でソート（数値型に変換）
- 転置：ファイル名@項目名が行、（年, 月）が列
- マルチインデックス化で階層構造を作成
- ファイル名でソート

### 出力例

```
                      (2024, 1)  (2024, 2)  (2024, 3)  ...
ファイル名  項目名
ボイラ      消費量       1000       1100       900
            単価          50         50         50
構外        消費量       500        520        480
油          消費量       200        210        190
...
```

## トラブルシューティング

### エラー：「ディレクトリが見つかりません」

```
エラー: ディレクトリ '../data' が見つかりません。
```

**対処法**：
- スクリプト実行ディレクトリを確認
- `directory` パスを絶対パスに変更

```python
directory = "/absolute/path/to/data"
```

### エラー：「'年' '月'不足」

**対処法**：
- ファイルに「年」「月」列が含まれているか確認
- 列名の正規化ルール（実績、タブ、改行除去）を確認

### エンコーディングエラー

**対処法**：
- `read_csv()` の `encoding` パラメータを調整
- UTF-8、UTF-16、CP932 など試行

```python
df = pd.read_csv(filepath, encoding='cp932')
```
